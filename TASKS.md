# Mira花礼 · 协作任务板

Coworker 在这里派活,Claude 执行。

## 工作流

1. **Coworker 派活**:在 [GitHub Issues](https://github.com/Francesca0518-source/MiraFleur/issues) 开一个 issue,描述任务
2. **Francesca / Claude 领活**:在 Issue 下面回复 "开始做了",创建分支 `issue/<number>-<slug>`
3. **Claude 执行**:改代码 → commit → 开 PR,PR 里写 `Closes #<number>`
4. **Coworker review**:GitHub PR 页面直接评论、请求修改,或 merge

## Issue 模板(建议 coworker 使用)

```
## 想要什么
<一句话说清楚目标>

## 为什么
<背景 / 用户场景>

## 验收标准
- [ ] ...
- [ ] ...

## 参考
<链接、截图、设计稿>
```
